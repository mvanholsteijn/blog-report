# blog-report
Generate a report on the number of blogs written based on an RSS feed

## Usage
```
Usage: blog-report [options]

  -u URL, --url=URL     	URL to RSS feed, default http://blog.xebia.com/feed/
  -f FEEDKEY, --feedkey=FEEDKEY	for RSS feed
  -m MONTH, --month=MONTH	 month to report on
  -y YEAR, --year=YEAR  	year to report on, default current
  -b, --print-blogs     	output a blog table
  -a, --print-authors   	output an author count
  -s, --print-story     	print a story
```
